# AI Learning

这是我的 AI 学习仓库。

它不是资料收藏夹，也不是搬运式笔记库，而是一个持续生长的学习系统：
- 用自己的话理解 AI
- 把学习过程沉淀成可复用的知识
- 同步到 GitHub，作为个人主页里 AI 技能树的一部分

我当前的目标不是追热点，而是把 AI 的底层逻辑、模型能力边界、系统组织方式真正学明白。最终，它会服务于一个更大的方向：构建能长期运行的 AI 系统，以及我自己的数字分身。

## 为什么这样学

我不想把 AI 学成一堆名词。

更重要的是回答清楚这几个问题：
- 模型是怎么学会东西的？
- 为什么大模型会表现出这么强的语言能力？
- 为什么它会幻觉、会犯错、会失效？
- 怎样把模型能力组织成一个真正能工作的系统？

所以这里的顺序不是“哪个火学哪个”，而是按下面这条线走：

地基 → 结构 → 系统

也就是：
- 先建立 AI 直觉
- 再理解机器学习训练逻辑
- 然后进入神经网络、Transformer、LLM
- 最后进入 RAG、Agent、多模态和系统设计
