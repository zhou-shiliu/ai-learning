# AI 学习路线图

> 起点：零基础。目标：建立完整 AI 认知体系，能独立理解模型、分析系统、追踪前沿，并最终走向 LLM / Agent / 数字分身。

---

## 总体思路

这条路线不是为了背名词，而是为了回答 3 个问题：

1. AI 是怎么从数据里学到规律的？
2. 大模型为什么会表现出这么强的语言能力？
3. 怎样把模型能力组织成一个能持续工作的系统？

所以学习顺序不追热点，而是按“地基 → 结构 → 系统”推进：

- 先建立 AI 的整体直觉
- 再理解机器学习的基本训练逻辑
- 然后进入神经网络、Transformer、LLM
- 最后进入 Agent、RAG、多模态和前沿方向

---

## 阶段划分

### Stage 1 · 直觉建立（0→1）
- 目标：先建立对 AI 的整体感觉，不急着做题，也不急着堆代码
- 核心问题：AI 到底是什么？它擅长什么？不擅长什么？为什么这几年突然爆发？
- 里程碑：能向一个完全不懂 AI 的人解释清楚“AI 和传统程序的区别”

### Stage 2 · 机器学习基础
- 监督学习 vs 无监督学习
- 线性回归
- 代价函数 / 损失函数
- 梯度下降
- 多元线性回归
- 逻辑回归
- 过拟合、偏差 / 方差、正则化

> 这一阶段看起来不“新”，但它是后面理解神经网络、Transformer、微调和模型边界的地基。

### Stage 3 · 神经网络基础
- 神经元 → 层 → 神经网络
- 前向传播与反向传播（先直觉理解，不急着硬背公式）
- 激活函数
- 神经网络为什么比线性模型更强

### Stage 4 · Transformer 与 GPT
- Attention 机制的直觉
- Transformer 架构为什么有效
- Token、Embedding、上下文窗口
- GPT 是怎么工作的（预训练 / 微调 / 对齐）

### Stage 5 · LLM 的能力与边界
- 上下文学习、提示词、工具调用
- 幻觉、推理失败、长上下文失效
- 常见使用模式与最佳实践
- RAG、结构化输出、评测

### Stage 6 · Agent 架构
- ReAct / Tool Use / Planning
- 记忆：短期 / 长期 / 知识检索
- 工作流编排
- 多 Agent 协作模式
- 从“会聊天”到“能做事”

### Stage 7 · 前沿与应用
- 多模态（图像、视频、语音）
- Long Context 与 RAG 深化
- 开源模型生态
- AI Safety / Alignment 基础
- 值得长期跟进的前沿方向

## 当前重点

当前主线不是泛泛了解 AI，而是先把“训练”这件事理解透。

眼下最重要的概念是：

- 监督学习 / 无监督学习
- 线性回归
- 代价函数
- 梯度下降
- 多元线性回归
- 逻辑回归
- 过拟合与正则化

这些内容看起来“老”，但它们是后面理解神经网络、Transformer、微调、幻觉、对齐和 Agent 系统的地基。
