# AI 学习路线图

> 起点：零基础。目标：建立完整AI认知体系，能独立分析AI系统、理解前沿进展。

## 阶段划分

### Stage 1 · 直觉建立（0→1）
- 目标：不做题、不写代码，先用具体例子建立对AI的直观感受
- 核心问题：AI到底是什么？它擅长什么？不擅长什么？
- 里程碑：能向一个完全不懂AI的人解释清楚"AI和程序的区别"

### Stage 2 · 神经网络基础
- 神经元 → 层 → 神经网络
- 前馈传播与反向传播（直观理解，不背公式）
- 激活函数、损失函数

### Stage 3 · Transformer 与 GPT
- Attention 机制直观的理解
- Transformer 架构为什么有效
- GPT 是怎么工作的（预训练 / 微调 / RLHF）

### Stage 4 · LLM 的能力与边界
- 上下文学习、思维链、工具调用
- 幻觉、推理失败、长上下文失效
- 常见使用模式与最佳实践

### Stage 5 · Agent 架构
- ReAct / Tool Use / Planning
- 记忆模块：短期 / 长期 / 知识检索
- 多 Agent 协作模式

### Stage 6 · 前沿与应用
- 多模态（图像、视频、语音）
- Long Context 与 RAG
- AI Safety 与 Alignment 基础
