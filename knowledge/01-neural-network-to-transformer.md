# 第一篇：神经网络与 Transformer——从细胞到 GPT

---

## 1. 先说直觉

AI 到底是什么？

我的理解：它是一个**极其复杂的函数**——你喂进去一堆数字，它吐出来另一堆数字。但这个函数的特殊之处在于，没人能解释它内部到底在干什么。它自己学会了。

这和传统编程不一样。传统编程是 if this then that，规则是人定的。神经网络是**数据喂出来的规则**，规则在权重里，没人直接写过。

---

## 2. 神经网络基础

### 什么是神经元

生物神经元：收到足够多的信号，就放电。

人工神经元（ perceptron）：收到一组数字（输入），乘以权重，加起来，通过一个激活函数，输出一个数。

```
输入 [x1, x2, x3]
权重 [w1, w2, w3]
加权求和 → 激活函数 → 输出
```

激活函数是什么？就是给输出加非线性，让神经元能表达复杂关系。没有它，一百层堆起来还是线性的。

常用激活函数：ReLU（大于0输出原值，小于0输出0）、Sigmoid（压缩到0-1之间）。

### 为什么要很多层

单层神经元能表达的东西很有限。堆多层，就有了层级抽象：

- 第一层学边缘
- 第二层学形状
- 第三层学物体

这就是深度学习"深度"的含义——层数越多，能表达的抽象层次越高。

### 训练到底是什么

训练不是编程，是**调参**。

用梯度下降：
1. 喂数据，算输出
2. 算错了多少（Loss / 损失）
3. 反向传播，问"每个权重对这个错误负多少责任"
4. 微调权重，往正确的方向走一步
5. 重复几百万次

最后这些权重就是"知识"。

## 3. Transformer 架构

### 它的出现解决了什么

在 Transformer 之前，RNN（循环神经网络）是主流。但 RNN 有一个根本问题：**顺序处理，无法并行**。句子从左到右读，后面的字要等前面的算完。

Transformer 用**自注意力机制（Self-Attention）** 解决了这个问题——所有位置可以同时计算。

### 自注意力机制——最核心的概念

它在做一件事：**每个词去看其他所有词，决定谁对谁重要**。

举例："The cat sat on the mat because it was tired."

这里的 "it" 指的是 cat 还是 mat？人一眼就知道是 cat。注意力机制让模型也学会这件事——"it" 应该多注意 "cat"。

数学上：
```
Attention(Q, K, V) = softmax(QK^T / √d) × V
```

- Q（Query）：我在找什么
- K（Key）：我包含什么特征
- V（Value）：我的实际内容

QK 做匹配，匹配结果乘以 V 得到输出。

### Transformer 的结构

```
输入 → 词嵌入 + 位置编码
        ↓
    多层编码器（Encoder）× N
        ↓
    多层解码器（Decoder）× N
        ↓
    输出（下一个词的概率分布）
```

- **词嵌入（Embedding）**：把文字变成数字向量。"猫" → [0.2, -0.5, 0.8, ...]
- **位置编码（Positional Encoding）**：Transformer 本身不看词的顺序，所以要把位置信息也喂进去

**编码器**（理解输入）：BERT 用编码器，做完形填空、情感分析。
**解码器**（生成输出）：GPT 用解码器，一个词一个词往外吐。

LLM（GPT 系列）基本只用解码器。

### FFN 与残差

每个 Transformer 层还有两个关键组件：

- **FFN（前馈神经网络）**：注意力之后，再做一次非线性变换
- **残差连接（Skip Connection）**："我先跳过这一层，直接把输入传过去，再加上这一层的输出"——让梯度流动得更好，训练更稳定

## 4. GPT 是什么

**Generative Pre-trained Transformer**。

三个词：
- **Generative**：能生成内容
- **Pre-trained**：在大规模数据上预训练过，不是从零开始
- **Transformer**：基于 Transformer 架构

GPT 的核心方法：**预测下一个词**。

喂给它一段话，它预测下一个词是什么。然后把预测出来的词再加进去，再预测下一个。一直重复，就生成了一整段话。

这个任务看起来简单——"下一个词有什么好预测的"——但当你预测了几千亿个词之后，模型涌现出了理解能力。

没人教它语法，它学会了语法。没人教它逻辑，它学会了逻辑。这就是**涌现（Emergence）**。

---

## 5. 我的理解

Transformer 真正的突破是**注意力机制**——它让模型学会"看全局"。每个词都能直接和每个词对话，不受距离限制。

这带来两个结果：
1. 长上下文成为可能（理论上无限，实际上有限）
2. 模型变得"慢思考"——它不是即时反应，而是要看完全局才能回答

这也解释了为什么 LLM 有"幻觉"——它生成的每个词都是"概率最高的那一个"，不是"唯一正确的那一个"。它不理解真假，它只理解统计。
