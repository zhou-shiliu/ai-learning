# 01 - 神经网络与Transformer

> 写于 2026-04-20，属于 Stage 1 → 直觉建立

---

## 核心问题

**AI（神经网络）和传统程序，到底有什么区别？**

传统程序：人写规则 → 计算机执行 → 输出确定结果
神经网络：人给数据 + 答案 → 计算机自己找规则 → 输出（可能是）对的答案

这是本质上的转变：从"告诉它怎么做"到"让它自己学出来"。

---

## 1. 神经元：信息的基本单元

想象一个小型决策器。

它收到几个数字（输入），每个数字乘以一个权重，然后把所有结果加起来，最后过一个激活函数（比如 ReLU：如果是负数就变成0，是正数就保留原值）。

这就是一个神经元干的事：**加权求和 → 激活 → 输出**

```
输入1(x1) --权重w1--> \
                      }→ (w1×x1 + w2×x2 + b) → 激活函数 → 输出
输入2(x2) --权重w2--> /
偏置 b（让决策边界可以平移）
```

例子：判断一张图片是不是猫

- 输入：每个像素的灰度值（比如 64×64 = 4096 个数字）
- 神经元做的事情：给每个像素配一个"猫特征权重"——有胡须轮廓的地方权重高，背景的地方权重低
- 输出：数字越大，越可能是猫

---

## 2. 层：神经元的排列方式

单个神经元能做的决策非常有限。真正的能力来自**层**。

- **输入层**：接收原始数据（像素、文本token等）
- **隐藏层**：层层处理，每一层都在把原始数据转换成更高层次的理解
  - 第一层：识别边缘和纹理
  - 第二层：识别形状
  - 第三层：识别眼睛、耳朵、尾巴
- **输出层**：给出最终判断（是猫 / 不是猫）

"层"这个概念对应了神经网络的**深度**——层数越多，能捕捉的特征就越抽象、越复杂。

---

## 3. 神经网络：如何找到正确的权重

初始状态：所有权重都是随机的（就像闭眼乱猜）

**训练过程（反向传播 + 梯度下降）：**

1. 送进去一张猫的图片，得到一个输出（比如 0.3，表示"不太像猫"）
2. 计算错误：正确答案应该是 1（100%是猫），输出是 0.3，差了 0.7
3. 反向传播：这个错误信号会一层层往回走，问每一层"你该为这个错误付多少责任"
4. 调整权重：每个权重都稍微移动一点，让错误变小一点
5. 重复这个过程——百万次、亿次之后，权重逐渐变成了"猫"的特征探测器

这就是为什么需要**大数据**：神经网络没有明确的规则，它靠统计规律在大量样本中逐渐雕刻出正确的权重分布。

---

## 4. Attention：让神经网络学会关注什么

传统神经网络处理序列（一句一句话）时，是一个字一个往后读的，经常忘记前面说了什么。

Attention（注意力机制）做了件事：**在任何位置做预测时，都可以回头看序列中的任意其他位置，并判断"我应该关注哪几个字"**。

例子：翻译 "The cat sat on the mat"

- 翻译到 "cat" 时，Attention 给 "cat" 很高的权重
- 翻译到 "sat" 时，Attention 同时关注 "cat" 和 "sat"（因为猫才会坐）
- 翻译到 "mat" 时，Attention 注意到 "on the"，知道这是在说"在……上面"

Attention 让模型能够捕捉长距离的语义依赖，而不用管它们之间隔了多远。

---

## 5. Transformer：把 Attention 用到极致

Transformer = 多层 Attention 的堆叠 + 残差连接 + LayerNorm + FFN

关键洞见：**只要注意力机制足够强，单靠它就能完成语言理解的大部分工作。**

GPT 就是这个架构的产物：
- 输入：一段文字
- 输出：下一个token的概率分布

GPT 不是"理解"了语言，而是通过海量文本的统计规律，学会了"在这个上下文里，下一个词最可能是什么"。

---

## 6. 直觉总结

| 概念 | 直觉类比 |
|------|----------|
| 神经元 | 一个加权投票器 |
| 层 | 逐层抽象的特征提取 |
| 反向传播 | 无数次小修正，从错误中学习 |
| Attention | 动态决定"此刻应该关注什么" |
| Transformer | 用 Attention 作为主力，抛弃了序列处理的顺序限制 |
| GPT | 超强的"下一个词预测器" |
